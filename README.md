
# Pokedex SPA

Aplicação front-end que consome a PokeAPI para exibir informações de Pokémon,
desenvolvida como uma SPA (Single Page Application) utilizando JavaScript puro e Vite.

#

## Funcionalidades

- Listagem de Pokémon consumindo API REST - (em construção)
- Busca por nome ou ID - (em construção)
- Renderização dinâmica no DOM - (em construção)
- Navegação sem recarregamento de página (SPA) - (em construção)
- Tratamento de loading e erros - (em construção)

#

## Tecnologias utilizadas

- HTML5
- SASS (SCSS)
- JavaScript (ES6+)
- Axios
- Vite
- Hot Reload integrado

#

## Organização do projeto

- Separação de responsabilidades (JS, SASS, HTML)
- Estilos organizados com SASS e partials
- Build e bundling com Vite
- Pasta public/ para assets estáticos

#

## Como rodar o projeto

### Clone do repositório:
```bash
git clone git@github.com:dvbcknd/pokedex-web.git
```

### Instale as dependências:
```bash
npm install
```

### Rode o projeto em modo desenvolvimento:
```bash
npm run dev
```
